Projeto acadêmico de um sistema web completo de gestão escolar, sectores Académico, Pedagógico e Financeiro.

Tecnologias: Python, Django, PostgreSql, Html, Css, Bootstrap.

O projeto de desenvolvimento do Sistema de Gerenciamento de Aprendizagem (LMS) tem como objetivo criar uma plataforma escalável para instituições educacionais, permitindo gerenciar cursos, alunos, professores e materiais didáticos. A equipe está focada em garantir a escalabilidade do sistema e a conformidade com a LGPD, utilizando serviços de nuvem (AWS) e tecnologias modernas para construção de uma arquitetura robusta. Este documento visa apresentar os planos de melhorias no processo, o plano de gerenciamento da qualidade e um relatório de lições aprendidas, com o objetivo de fornecer uma visão completa do status do projeto e das práticas implementadas para alcançar os objetivos definidos.
